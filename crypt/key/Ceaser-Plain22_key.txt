Plaintext: We have discussed the use of hash functions for message authentication and for the creation of digital signatures. Here are is an examples of secure hash function application. Passwords. In this scheme a hash of a password is stored by an operating system rather than the password itself. Thus the actual password is not retrievable by a hacker who gains access to the password file. In simple terms when a user enters a password the hash of that password is compared to the stored hash value for verification. This application requires preimage resistance and perhaps second preimage resistance.
